Tutorial install dan penggunaan sistem SSP-KMU
1. Copy file ke dalam folder htdocs pada xammp
2. Import database siskop_db.sql ke dalam localhost/phpmyadmin
3. Untuk menjalankan sistem ketikkan "localhost/siskop/public/" atau "localhost/siskop/public/index.php"
4. untuk login gunakan username : supriadi@gmail.com , dan password : 12345678
5. Menu Beranda akan menampilkan dashboard dan halaman utama sistem. 
6. Menu Anggota akan menampilkan data anggota dan mengelola data anggota koperasi
7. Selesei.
